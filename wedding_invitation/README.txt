
# Инструкция по загрузке сайта на GitHub Pages

1. Перейдите на https://github.com и создайте новый публичный репозиторий, например: `wedding-invitation`.

2. Загрузите содержимое этой папки в репозиторий:
   - `index.html`
   - Папку `img/` с изображениями

3. Перейдите в настройки репозитория:
   - Откройте вкладку `Settings`
   - Найдите раздел `Pages`
   - В разделе Source выберите ветку `main` и папку `/ (root)`
   - Нажмите `Save`

4. Через несколько секунд сайт станет доступен по ссылке:
   `https://<ваш-логин>.github.io/wedding-invitation/`

5. Чтобы использовать персонализированное обращение, добавьте к ссылке `?name=Имя`
   Пример: `https://example.github.io/wedding-invitation/?name=Ольга`
